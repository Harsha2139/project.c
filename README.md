1 ABSTRACT

The Citizen Records Management System is a C-based console application designed to store, manage, and retrieve citizen information efficiently. It allows the user to add new citizen records, view all records, search citizens by name, view details using an ID, delete unwanted records, and save/load data from a file. The system is useful for maintaining organized data in small organizations, communities, or educational projects.

3.TECHNICAL REQUIREMENTS

Hardware Requirements:

Any basic computer or laptop

Minimum 2GB RAM

100MB free storage

Software Requirements:

C Compiler (GCC, Turbo C, MinGW, etc.)

Any IDE (CodeBlocks, VS Code, DevC++, Turbo C++, etc.)

Operating System: Windows / Linux / macOS

4. FUNCTIONAL REQUIREMENTS

The system shall allow adding new citizen records with fields:

ID (auto-generated)

Name

Age

Address

Phone number

The system shall store citizens in an array (up to 500 records).

The system shall allow:

Displaying all citizens

Searching by name

Viewing full record by ID

Deleting a record based on ID

The system shall save all data in a .dat file.

The system shall load the saved data automatically when the program starts.
